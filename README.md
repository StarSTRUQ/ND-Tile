# ND-Tile

Tile a scalar field in an N-dimensional independent variable space using N-dimensional planar fitting.

## Usage

To import Tiling, just add the ND-Tile directory to your PYTHONPATH environment variable.

## Dependencies

* Python 3

* numpy

* scipy
